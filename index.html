Prompt Profesional Optimizado

Este prompt fue diseñado para generar aplicaciones modernas con Next.js + TypeScript + TailwindCSS + Firebase, optimizadas para despliegue en Vercel y control de versiones en GitHub.

Objetivo General

Desarrollar una aplicación web profesional, moderna, rápida y responsive, lista para producción y sin errores comunes de compilación o despliegue.

Requisitos Técnicos

  • Next.js 14+
  • TypeScript
  • TailwindCSS
  • Firebase Modular SDK
  • ESLint + Prettier
  • Arquitectura modular

Estructura Recomendada

/app
/components
/lib
/hooks
/services
/styles
/types
/utils
/public

Variables de Entorno

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=

Buenas Prácticas

  • No usar JavaScript inline
  • No usar alert() o confirm()
  • Evitar hydration mismatch
  • Separar componentes y lógica
  • No exponer credenciales
  • Usar TypeScript estricto

Optimización para Vercel

{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}

Diseño UI/UX

  • Diseño minimalista y moderno
  • Modo oscuro premium
  • Responsive móvil primero
  • Animaciones suaves
  • Accesibilidad AA

Deploy Profesional

La aplicación debe quedar lista para:

  • Subir a GitHub
  • Conectar con Vercel
  • Agregar variables ENV
  • Desplegar sin modificaciones adicionales
